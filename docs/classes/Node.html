Node

Defined in src/main/javascript/node.js:11

The initializer for a scene graph node.

Parameters:

  • parameters Object

    an object with optional information to include in the node. The possibilities are:

    • name {string}: nodes can be named if they need to be retrieved later.
    • transform {Float4x4}: a transformation matrix to apply before drawing or traversing children.
    • state {function}: a parameter-less function to call before drawing or traversing children. this function may set any render state needed.
    • shape {string}: the name of a shape to draw.
    • children: the node will have an array of children by default, but if the node is intended to be a leaf, you can save the space and time associated with an empty list by setting this value to "false".
