    Shader

    Defined in src/main/javascript/shader.js:218

    static method to create and construct a new Shader.

    Parameters:

    • name String

      name to retrieve this shader

    • vertexShaderUrl String

      url to the vertex shader GLSL file

    • fragmentShaderUrl String

      url to the fragment shader GLSL file

    • attributeMapping Object

      maps POSITION, NORMAL, and TEXTURE attributes to the attribute names in the shader. This allows the engine to manage the attributes without forcing the shader author to use "standard" names for everything. Defaults to:

      • POSITION_ATTRIBUTE: "inputPosition"
      • NORMAL_ATTRIBUTE: "inputNormal"
      • TEXTURE_ATTRIBUTE: "inputTexture"
    • parameterMapping Object

      maps standard parameters to the parameter names in the shader. This allows the engine to manage setting the standard set of parameters on the shader without forcing the shader author to use "standard" names. Defaults to:

      • MODEL_MATRIX_PARAMETER: "modelMatrix"
      • VIEW_MATRIX_PARAMETER: "viewMatrix"
      • PROJECTION_MATRIX_PARAMETER: "projectionMatrix"
      • NORMAL_MATRIX_PARAMETER: "normalMatrix"
      • OUTPUT_ALPHA_PARAMETER: "outputAlpha"

    Returns:
