      url to the vertex shader GLSL file

    • fragmentShaderUrl String

      url to the fragment shader GLSL file

    • attributeMapping Object

      maps POSITION, NORMAL, and TEXTURE attributes to the attribute names in the shader. Defaults to:

      • POSITION_ATTRIBUTE: "inputPosition"
      • NORMAL_ATTRIBUTE: "inputNormal"
      • TEXTURE_ATTRIBUTE: "inputTexture"
